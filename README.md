# Atividade-codigo-alta-perfomace-1903# Dreamy Donuts - Vitrine Digital Interativa

Este projeto foi desenvolvido como parte de uma atividade prática de *Digitalização do Comércio Local*. O objetivo é profissionalizar a venda de donuts artesanais, facilitando a escolha do cliente e a organização dos pedidos para o vendedor.

---

##  O Problema de Negócio (Defesa)

Muitas confeitarias artesanais perdem tempo atendendo pedidos de forma fragmentada pelo WhatsApp, tendo que explicar sabores, preços e somar valores manualmente para cada cliente. Isso gera lentidão e erros humanos no fechamento da conta.

*A Solução:* Esta landing page funciona como uma *Vitrine Digital*. O cliente visualiza os donuts disponíveis, seleciona as quantidades e vê o valor total sendo calculado em tempo real. Ao finalizar, o pedido é enviado já formatado para o WhatsApp da loja, economizando tempo e passando uma imagem muito mais profissional para a marca.

---

##  Requisitos Técnicos Atendidos

* *Estrutura Semântica (HTML5):* Uso de tags como <header>, <main>, <article> e <footer> para uma página bem estruturada.
* *Estilo Responsivo (CSS3):* Design *Mobile First*, garantindo que o site funcione perfeitamente em qualquer tamanho de tela (celular, tablet ou PC).
* *Lógica de Programação (JavaScript):*
    * Manipulação de eventos de clique (addEventListener).
    * Cálculo dinâmico de subtotais e total geral.
    * Integração com a API do WhatsApp para envio de mensagens automáticas.

---

## identidade Visual

A interface foi inspirada em uma paleta de cores "Candy Colors" (tons de rosa e marrom chocolate), com um toque lúdico inspirado em animações clássicas (ícone de donut estilo Simpsons no cabeçalho), criando um ambiente acolhedor e atrativo para o consumidor.

---

## Como utilizar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo index.html em seu navegador.
3. Escolha os sabores, verifique o total no rodapé e clique em "Pedir pelo WhatsApp".
